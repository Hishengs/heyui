1.引用

这是一段引用...span.h-quote

2.对齐

居左对齐 h-quote.text-left

居中对齐h-quote.text-center

居右对齐h-quote.text-right

3.显示-display

block

span.dis-block

inline-block

div.dis-inline-block

4.文本大小

text-xxxlarge

text-xxlarge

text-xlarge

normal

text-small

5.文本颜色

主要: text-primary/blue

成功: text-success/green

信息: text-info/sky-blue

警告: text-warning/orange

危险: text-danger/red

6.背景颜色

主要: bg-primary/blue
成功: bg-success/green
信息: bg-info/sky-blue
警告: bg-warning/orange
危险: bg-danger/red

7.hide 显示/隐藏

通过添加show,hide,invisible来显示或隐藏元素，其中hide既不显示元素也不占用文档流位置

8.浮动 float (使用.clear-float来清除浮动)

左浮动 右浮动

9.内外边距

margin和padding分别提供了4种距离：10 15 20 30
通过设置class: margin-top/right/bottom/left-x 和 padding-top/right/bottom/left-x
